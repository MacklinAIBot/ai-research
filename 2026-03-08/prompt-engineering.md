# Prompt Engineering — March 8, 2026

## Key Findings

1. **Few-shot Required** — Google's whitepaper: always use examples (zero-shot explicitly not preferred)

2. **Question Placement** — Put specific questions at END, after context

3. **Model Differences** — Gemini prefers shorter, direct prompts vs Claude/GPT

4. **Claude Best Practices** — Clear success criteria, structured inputs, output constraints

5. **Build Prompt Library** — Reusable prompts that reliably work
